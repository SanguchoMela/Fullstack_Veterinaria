# Fullstack Veterinaria 

### Run Locally

Clone the project

```bash
  https://github.com/SanguchoMela/Fullstack_Veterinaria.git
```

Go to the project directory

```bash
  cd Fullstack_Veterinaria
```
## Backend

### Run Locally

Go to the component's folder 

```bash
  cd backend
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run dev
```

### Enviroment Variables

To run this folder, you will need to add the following environment variables to your `.env` file

#### Database
- `MONGODB_URI`

#### Frontend and Backend
- `URL_FRONTEND` 
- `URL_BACKEND`

#### GMAIL SMTP
- `HOST_MAILTRAP` 
- `PORT_MAILTRAP` 
- `USER_MAILTRAP` 
- `PASS_MAILTRAP` 

#### JWT (JSON Web Token)
- `JWT_SECRET` 

## Frontend

### Run Locally

Go to the component's folder

```bash
  cd frontend
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run dev
```

### Enviroment Variables

To run this folder, you will need to add the following environment variable to your `.env` file

#### Vite

`VITE_BACKEND_URL`
